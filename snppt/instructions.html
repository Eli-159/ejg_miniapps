Student Notices PowerPoint App Instructions


Installation

The SNPPT App runs mostly online, but it is helpful if you have a few bookmarks set. (This works best in Chrome)


  1. Create a folder in your bookmarks bar called 'Notices'.


  2. Add a bookmark to this folder called 'Student Notices'. Enter the following url where asked:

    https://qedu.sharepoint.com/sites/1668/Student/Lists/StudentNotices/Current_Notices.aspx?viewid=84ba931b%2D859d%2D4f84%2D8a32%2De48db687cd77



  3. Add a second bookmark called 'Copy Notices'. This time, add the following code to the url box.



  4. Finally, add one last bookmark called 'SNPPT App' with the following url.

    https://snppt-app-5546u.ondigitalocean.app/?#




Using It


  1. Go to the 'Student Notices' page

  2. Once this page has loaded, click the bookmark saved as 'Copy Notices'. If it is successful, it will display a message saying so and you can proceed. Otherwise, reload the page and try again.

  3. Go to the 'SNPPT App' page. Paste the data as per instructions on the page.

  4. Select your grade and click the 'Submit' button.

  5. From here, just follow instructions on the page to create a PowerPoint or display the data in your browser.
