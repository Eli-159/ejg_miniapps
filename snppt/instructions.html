Installation

The SNPPT App runs mostly online, but it is helpful if you have a few bookmarks set. (This works best in Chrome or Edge)


  1. Open your browser. Make sure your Bookmarks Bar is open. In Chrome, this can be toggled using Ctrl+Shift+B. To do this is Edge, you will need to click on the star icon with three lines through it at the top of the screen, click on the three dots in the window that appears and then select 'Show Bookmarks Bar'. Alternativly in Edge, you can just click on the star icon with three lines through it each time.

  2. Create a folder in your bookmarks bar called 'Notices'. This can be done by right-clicking your bookmarks bar and selecting 'Add Folder'.


  3. Add a bookmark to this folder called 'Student Notices'. In Chrome, this can be done by right clicking on the folder icon of your folder and selecting 'Add Page'. If you are using Edge, you will need to go to the Bookmarks Menu, (the star icon with three lines through it) add the current tab as a bookmark and then right click on it, select edit, and update the name and url. Copying the Data


  1. Go to the 'Student Notices' page

  2. Once this page has loaded, click the bookmark saved as 'Copy Notices'. If it is successful, it will display a message saying so and you can proceed. Otherwise, reload the page and try again.

  3. Go to the 'SNPPT App' page. Paste the data as per instructions on the page.

  4. Select your grade and click the 'Submit' button.



Editing the Data


  1. Copy and submit the data on the SNPPT page.

  2. Select the 'Edit Data' option.

  3. Adjust the data as desired. The data points for each notice are as follows:
    • Display: Whether or not the notice is shown in the PowerPoint and View Data table.
    • Category: The category the notice was in. This cannot be changed.
    • Subject: The custom heading of the notice.
    • Message: The main body of the notice.
    • Teacher: The teacher that posted the notice. This cannot be changed.
    • Highlight: Whether or not the notice is highlighted when printed from the View Data table. This defaults to whether or not the notice was new.

  4. Select the 'Save' option.
